# Guida introduttiva a PythonOS

---

## Installazione

Digitate nel terminale del Desktop:

<div class="terminal"><div class="terminal-top">
<div class="dot red"></div>
<div class="dot yellow"></div>
<div class="dot green"></div>
</div><div class="terminal-body">> git clone https://github.com/pythonos-project/pythonos.git<br>
> python -m venv venv<br>
> source venv/bin/activate<br>
<em># Su Windows: venv\Scripts\activate</em><br>
<em># PowerShell: venv\Scripts\Activate</em><br>
> pip install -r requirements.txt<br>

</div></div>Così abbiamo installato PythonOS nel Desktop e tutte le dipendenze necessarie.

---

## Avvio

Digita:

<div class="terminal"><div class="terminal-top">
<div class="dot red"></div>
<div class="dot yellow"></div>
<div class="dot green"></div>
</div><div class="terminal-body">> python main.py<br>

</div></div>Vedrai una lockscreen con un PIN.

Il PIN predefinito è 1234.

Dopo averlo inserito vedrai l'interfaccia di PythonOS.
